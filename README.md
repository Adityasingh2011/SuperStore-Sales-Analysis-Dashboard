# SuperStore Sales Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using SuperStore sales data to analyze business performance, profitability, customer segments, and regional sales trends. The dashboard helps stakeholders make informed decisions through data-driven insights and visual analytics.

## Objectives

* Analyze sales and profit performance
* Identify top-performing products and categories
* Evaluate regional and customer segment trends
* Discover opportunities for business growth
* Support decision-making through interactive visualizations

## Tools Used

* Power BI
* Excel
* CSV
* Data Visualization

## Dashboard Highlights

* Sales & Profit Overview
* Regional Performance Analysis
* Category & Sub-Category Insights
* Customer Segment Analysis
* Top Products Performance
* Interactive Filters and KPIs

## Business Impact

The dashboard provides actionable insights into sales trends, profitability, and customer behavior, enabling businesses to optimize strategies and improve overall performance.
